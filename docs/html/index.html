class src.CWSpc(FileName)¶

parent class of 1D and 2D CWSpc

Parameters

  • B – n-by-1 float matrix. field in undefined unit

  • spc – n-by-m float matrix, where m >= 1 is the size of the second dimension. spectrum/spectra intensity.

  • acqParams – struct of acquisition parameters, including MW power, receiver gain, modulation amp, time constants, conversion time

  • NX – size of the 1st dimension

  • NY – size of the 2nd dimension

  • is2d (is1d,) – type of CWSpc

  • NScan – number of scan

subtractbg(cwspc)¶

subtract background spectrum. If cwspc is 1D, it asserts the acq params are same. If cwspc is 2D, it takes mean of the 2nd dimension, and asserts the acq params are same.
